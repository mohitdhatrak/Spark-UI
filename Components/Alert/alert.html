How to modify component as per need !

All alerts have some default properties like background-colour, colour, etc. If you want a alert of your custom colour, use the below given css-variabes in :root selector of your CSS file. No need to use these variables if you wish to use the alert as it is. For the given cross icon to be visible, you need to link the given google-icons stylesheet in your HTML too.

1. Simple info alert

Your response was recorded!


By default the alert will be displayed at the bottom of the page, as class alert-center is used in the outermost < div > . To change position of alert to bottom right, replace class alert-center with alert-right. To change position of alert to left, replace class with alert-left. Disclaimer: Make sure you don't use this given HTML of alert component inside any container having position: relative; as alert will get positioned according to the parent container. So ideally parent container should be body.
