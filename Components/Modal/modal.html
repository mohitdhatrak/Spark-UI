How to modify component as per need !

The modal component has many defaults like width, colors, borders-styles, icon-size, etc. For changing any of these styles as per your need, just use the below given css-variables in :root selector of your CSS file. You can change the width of the modal by given --modal-width variable. No need to use these variables if you wish to use the modal as it is.

3. Simple modal

Choose any one

This is a simple modal where you directly select the option, no confirmation buttons. If you have many options, you can give the component a fixed height, using css-variable --simple-modal-height. This will automatically give it a scrollbar as shown (here height given is 12rem). If you have lesser options or want no scrollbar, put this value --simple-modal-height: max-content;. For the given icon to be visible, you need to link the given google-icons stylesheet in your HTML too. You can replace or remove icons too.
