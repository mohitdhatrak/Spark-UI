How to modify component as per need !

Both images have no border by default. For adding borders, just use the given css-variable below in :root selector of your CSS file. For changing the sizes, css-variables of width are given for both images. No need to use these variables if you wish to use the images as it is.

1. Responsive Image

The above image is responsive, you can set any required width in the given css-variable --responsive-image-width.

2. Small Round Image

This round image is not responsive, hence it is recommended to provide a small width that can even fit a small screen (css-variable --round-image-width). It crops image appropriately to make it round, while maintaining the aspect ratio.
