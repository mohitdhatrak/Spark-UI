How to modify component as per need !

Both available rating star components - 'outline-only' and 'filled' have default values for background-color, icon-color, icon-size, hover-color, gap between stars. For changing these as per your choice, just use the given css-variable below in :root selector of your CSS file (unique variables for both types so you can use both together too). No need to use these variables if you wish to use the component as it is.

Both rating star types - 'outline-only' and 'filled' have same HTML code. The given HTML code is for the 'outline-only' rating component, so the class used is rating-outline.
