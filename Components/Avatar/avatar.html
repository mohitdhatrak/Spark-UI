How to modify component as per need !

All available avatar types have no border by default. For adding borders, just use the given css-variable below in :root selector of your CSS file. As for the choice in sizes, there are 5 options already available, how to access them is given below. If you want a size other than those, then use the given css-variable. No need to use these variables if you wish to use the avatars as it is.

Standard avatars

All 5 sizes have the same HTML code given below. The given HTML code is for the medium size, therefore the class used is avatar-medium. To use a larger size than medium, replace class avatar-medium with avatar-large. To use the largest size, replace class with avatar-extra-large. To use a smaller size than medium, replace class with avatar-small. To use the smallest size, replace class with avatar-extra-small.
