How to modify component as per need !

All available cards have some default width and other properties like colour, border, etc. All cards are responsive, to give width as per your convenience, just put the given html in a <div> and give that <div> the width you wish. For changing colours, borders, box-shadow, on hover-effects, icon-sizes, just use the given css-variables below in :root selector of your CSS file. By default there is box-shadow effect on card hover. To get box-shadow even without hover, use given css-variable --card-box-shadow. No need to use these variables if you wish to use the card as it is.
