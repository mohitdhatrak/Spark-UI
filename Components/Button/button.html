How to modify component as per need !

All 3 categories of buttons (primary, secondary, warn) have some default properties like colour, background-color, border, etc. For changing colours, borders, just use the given css-variables below in :root selector of your CSS file. The reason for providing separate css-variables for all 3 categories of buttons is, if you wish to use any 2 or all 3 categories together, you can customize each button as per your requirement i.e. all 3 can have different border styles or colour. No need to use these variables if you wish to use the buttons as it is. For the given icons inside buttons to be visible, you need to link the given google-icons stylesheet in your HTML too.

Google-icons CSS - HTML link tag

1. Standard buttons


There are 3 categories of buttons - primary, secondary and warn button. By default, HTML code for primary button is provided, hence class used is button-primary. To use secondary button, replace class button-primary with button-secondary. To use warn button, replace class with button-warn. All these buttons can have custom styles and colours.

2. Link buttons

Page 1 Page 2

There are 2 categories of link buttons - primary and secondary. By default, HTML code for primary link button is provided, hence class used is link-primary. To use secondary link button, replace class link-primary with link-secondary. Both of these buttons can have custom styles and colours.

3. Icon buttons


There are 3 categories of icon buttons - primary, secondary and warn button. By default, HTML code for primary icon button is provided, hence class used is button-primary. To use secondary icon button, replace class button-primary with button-secondary. To use warn icon button, replace class with button-warn. All these buttons can have custom styles and colours.

4. Floating action buttons


There are 3 categories of floating action buttons - primary, secondary and warn button. By default, HTML code for primary button is provided, hence class used is button-primary. To use secondary button, replace class button-primary with button-secondary. To use warn button, replace class with button-warn. All these buttons can have custom styles and colours, except here border-width is fixed as increasing border-width distorts the circular shape.
