How to modify component as per need !

The available navigation components have a few default properties like background-color, color, border. For changing colours, borders, box-shadow, on hover-effects, icon-sizes, just use the below given css-variables in :root selector of your CSS file. No need to use these variables if you wish to use the navigation as it is.
