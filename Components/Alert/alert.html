How to modify component as per need !

All alerts have some default properties like background-colour, colour, etc. If you want a alert of your custom colour, use the below given css-variabes in :root selector of your CSS file. No need to use these variables if you wish to use the alert as it is.

1. Simple info alert

Your response was recorded!


By default the alert will be displayed at the bottom-right of the page, as class alert-right is used in the outermost <div>. To change position of alert to bottom left, replace class alert-right with alert-left. This component is also responsive, it adjusts width according to screen-size.

2. Success/Error alert

Login request successful!

Login attempt failed!


Above simple alert component instructions for position are valid here too, make sure you read those. By default you will get the green coloured success alert in below given HTML code. To use the red error alert, replace class alert-success with class alert-error.
